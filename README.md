node-train
---

Query things about Chinese railway via CLI.

## Install

```
[sudo] npm install cntrain [-g]
```

## Usage

- Query train schedule

```sh
cntrain -n g1
```

- Look for help

```sh
cntrain -h
```

- Get the vertion

```sh
cntrain -V
```
